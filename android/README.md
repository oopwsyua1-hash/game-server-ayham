# Hani Jar Clone - Android App

## 📱 التطبيق الأساسي

تطبيق Android احترافي لـ Hani Jar Clone مبني بـ **Kotlin** و **Jetpack Compose**.

## ✨ الميزات

- ✅ نظام تسجيل الدخول والتسجيل
- ✅ قائمة الغرف
- ✅ إنشاء غرف جديدة
- ✅ نظام الهدايا
- ✅ الملف الشخصي
- ✅ الترتيبات والإحصائيات
- ✅ التواصل الفوري مع Socket.io

## 🛠️ المتطلبات

- Android Studio
- Java 8+
- Gradle 8.0+

## 📦 المكتبات المستخدمة

- Retrofit2
- OkHttp3
- Socket.io
- Coroutines
- Glide
- Material Design

## 🚀 البدء

```bash
cd android
./gradlew build
```

## 📝 الملاحظات

- تأكد من تحديث `BASE_URL` في `RetrofitClient.kt` مع رابط الـ Backend الصحيح
- أضف `google-services.json` إذا كنت تستخدم Firebase
