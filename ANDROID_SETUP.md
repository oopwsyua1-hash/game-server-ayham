# 📱 دليل إعداد تطبيق Android باستخدام Capacitor\n\n## المتطلبات:\n\n### 1. **Node.js**\n```bash\nnode --version\nnpm --version\n```\n\n### 2. **Java JDK 11+**\n```bash\njava -version\n```\n\n### 3. **Android Studio**\n- حمّل من: https://developer.android.com/studio\n\n### 4. **Git**\n```bash\ngit --version\n```\n\n---\n\n## ✅ خطوات التثبيت:\n\n### **الخطوة 1: تحضيرات أولية**\n\n```bash\ngit clone https://github.com/oopwsyua1-hash/game-server-ayham.git\ncd game-server-ayham\nnpm install\n```\n\n### **الخطوة 2: إعداد Capacitor**\n\n```bash\n# على Linux/Mac\nchmod +x CAPACITOR_SETUP.sh\nbash CAPACITOR_SETUP.sh\n\n# أو يدويّاً\nnpm install @capacitorjs/core @capacitorjs/cli @capacitorjs/android\nnpx cap add android\nnpx cap sync\n```\n\n### **الخطوة 3: فتح Android Studio**\n\n```bash\nnpx cap open android\n```\n\n### **الخطوة 4: تشغيل على الهاتف**\n\n1. وصّل الهاتف بـ USB\n2. فعّل USB Debugging (Settings → Developer Options)\n3. اضغط الزر الأخضر ▶️ في Android Studio\n\n---\n\n## 🔧 أوامر مهمة:\n\n```bash\n# مزامنة الملفات\nnpx cap sync\n\n# فتح Android Studio\nnpx cap open android\n\n# بناء APK\ncd android && ./gradlew assembleDebug && cd ..\n```\n\n---\n\n**لمزيد من التفاصيل، راجع QUICK_START.md** 📖\n
