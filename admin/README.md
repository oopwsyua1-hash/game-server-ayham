# Hani Jar Admin Dashboard

## 🎛️ لوحة التحكم الإدارية

تطبيق React احترافي لإدارة منصة Hani Jar مع جميع الميزات اللازمة.

## ✨ الميزات

- ✅ لوحة تحكم شاملة
- ✅ إدارة المستخدمين
- ✅ إدارة الهدايا
- ✅ إدارة الغرف
- ✅ إدارة الوكلاء
- ✅ التقارير والإحصائيات
- ✅ نظام مصادقة آمن

## 🚀 البدء

```bash
cd admin
npm install
npm start
```

## 🛠️ التكنولوجيات المستخدمة

- React 18
- Material-UI (MUI)
- Axios
- Recharts
- React Router

## 📝 الملاحظات

- تأكد من تشغيل Backend قبل تشغيل Dashboard
- استخدم بيانات اعتماد المسؤول للدخول
